Please address bugs and your suggestions to Github issues or E-mail.

Basic usage

$('#basic').scroller();
<section id="basic">
	<ul>
		<li><img src="image-1.jpg" ... /></li>
		<li><img src="image-2.jpg" ... /></li>
		<li><img src="image-3.jpg" ... /></li>
		<li><img src="image-4.jpg" ... /></li>
		<li><img src="image-5.jpg" ... /></li>
	</ul>
</section>

Change speed

$('#speed').scroller({ step:15 });

Vertical scrolling

$('#vertical').scroller({ direction: 'vertical' });

Tested in browsers:

+ Firefox: 3.5.19, 5.0

+ Opera: 11.50

+ Google Chrome: 12

+ Internet Explorer: 6, 8, 9

+ Safari: 5.0.5

Dependencies:

Required jQuery 1.3+

Installation:

In <head> tag:

<link href="path/to/jquery.scroller.css" type="text/css" rel="stylesheet" />
<script src="path/to/jquery.scroller.js" type="text/javascript"></script>

JavaScript:

$(function(){
	$('selector').scroller();
});
